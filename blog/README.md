## SSG를 사용하면 좋은 페이지
```
데이터가 많이 바뀌지 않을 페이지에 좋다!

사용자가 페이지를 요청하기 전에 pre-render 할 수 
있다면 SSG
없다면 SSR 또는 ISR 또는 CSR
```
- Marketing pages
- Blog posts
- E-commerce product listings
- Help and documentation (고객센터, 문서)
